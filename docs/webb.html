Vad är det som jag gjort och lärt mig?

Det som jag har gjort är att lära mig bland annat hur man använder sig av HTML och CSS för att skapa webbsidor som denna. HTML användes för att skapa själva informationen som ska visas på webbsidan medans CSS används för att styla de element som HTML ger genom att linka ihop dem.

Inom HTML så kan man till exempel lägga till text som man kan ge olika klasser för att kunna styla dem på olika sätt inom css, även så går det att lägga till bilder. Inom CSS så kan man ändra färg på bakgrund, text och lägga till saker som fonts och positionera element. Man kan även animera element genom att använda keyframes funtkionen där man sätter olika stadium för ett element att vara i som den går imellan, du kan se det "in action" här under!

Github har används till att spara projektet och även används för att kunna göra så att alla kan ha tillgång till både koden och webbsidan själv.
