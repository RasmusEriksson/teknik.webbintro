Historia om hur webben uppstod

Webben uppstod först i syfte där flera forskare runt om i världen ville kunna automera att sprida information till både andra forskare och olika institut. Det var då när den britiska forskaren Berners-Lee som jobbade på CERN förslog den första itterationen av en webbsida och en server. Det togs fram som ett hypertext project (html) där det enligt Cerns egna webbsida beskrivs som "a web of hypertext documents" där "browsers" kan se olika hypertext dokument. Namnet på hela projectet kallades för "World Wide Web" eller "WWW" förkortat.
